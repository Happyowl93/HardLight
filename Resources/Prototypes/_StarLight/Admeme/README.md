# Admeme Prototypes
Putting this readme here to explain the existence of this directory's existence.

Supposedly Darkrell has instructed future admeme prototypes to be placed into this Admeme directory. I was told that this was stated in #staff-dev but I don't have access to that channel so I can't actually see that for myself.
